## Principles

- **Correctness first.** Financial/analytical logic must be tested and
  reproducible before it is trusted.
- **Reproducibility.** Deterministic pipelines; data and configuration are
  versioned or referenced, never implicit.
- **Safety & reversibility.** Prefer designs that can be paused, audited, and
  rolled back.
- **Simplicity.** Prefer the simplest design that works; add complexity only
  when justified.
- **Security.** Secrets never in Git; least-privilege access to any external
  service.
